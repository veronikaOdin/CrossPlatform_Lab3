# Лабораторна робота №3
Дисципліна: "Кросплатформне програмування"
Виконала: Одинець Вероніка

## Опис проєкту
Проєкт містить вирішення Завдання №0:
* `Task0SAX.java` — аналіз XML-документа за допомогою SAX-парсера.
* `schema.xsd` та `XMLValidator.java` — XSD-схема та програмний валідатор.
* `Task0DOM.java` — вибірка, сортування за рейтингом та генерація нового XML-файлу через DOM-парсер.
